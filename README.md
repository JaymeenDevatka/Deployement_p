# 📝 Notes App — Backend Service  
A modern, containerized Notes/Todo API built with **Flask + Gunicorn + Docker**.  
Ideal for DevOps portfolios using **CI/CD (Jenkins), Kubernetes, and AWS**.

---

## 🚀 Features  
- REST API for Todo CRUD  
- Health Check endpoint  
- Clean Python architecture  
- Dockerized (production-ready)  
- Ready for Jenkins CI/CD pipelines  
- Ready for Kubernetes deployment
